基本使用步骤 ： 创建文件 ： mkdir 文件名 进入文件 ： $cd 文件名 1、初始化仓库 git init 2、添加到暂存区 git add 文件名 绿色 3、提交到仓库 git commit -m"记录信息" -m后面记录本次提交说明 第一次提交报错 补全信息 git config --global user.email "Z18945417391@163.com" git config --global user.name "zhangxy2" 4、查看记录 git log 5、查看没被提交的文件 git status 6、全部提交 ： git add . 当前目录和 后代目录文件 git add all 所有都提交 注意 ： git add . 空格 git commit -m "" 空格 用户名 密码 设置一次就可以 以后不需要再次设置 修改
